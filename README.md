**¡Una app divertida para sortear amigos secretos!**  

## 🤔 ¿Cómo usar?🤔  
1. Escribe los nombres de tus amigos en el campo.  
2. Haz clic en **"Añadir"**.  
3. Cuando tengas todos, presiona **"Sortear amigo"**.  
4. ¡Descubre quién es tu amigo secreto!
